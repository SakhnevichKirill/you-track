## Description

Hard deadline value refresh using values from "Start date" and "Original esimation" (in my case: "Hard interval").

## Installation

```bash
$ npm install
```
Create file .development.env and add:
```bash
TOKEN="perm:your-token"
BASE_URL="http://example.myjetbrains.com"
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
